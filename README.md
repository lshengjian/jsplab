常用快捷键
Ctrl + D：选中下一个匹配项
Shift + Alt + F：格式化代码
F2：重命名变量或元素
Ctrl + Home / Ctrl + End：快速定位到文件顶部或尾部
Ctrl + Backspace / Ctrl + Delete：删除前一个或后一个单词
Ctrl + Shift + K：删除整行
Alt + ↑ / ↓：向上或向下移动行
Shift + Alt + ↑ / ↓：向上或向下复制行
多行操作方法
按住 Alt 键，用鼠标左键点击多处，出现多个光标。
Ctrl + Alt + 向上/向下箭头：在多行上出现光标。
Shift + Alt + I：在选中行的每行末尾出现光标。
Ctrl + D：选中下一个匹配项。
Shift + Ctrl + L：选中所有匹配项。
